# 📊 Customer Churn Analysis (Excel)

Brief analysis of customer churn using **Microsoft Excel** to identify key drivers behind customer attrition.

## 🔍 Key Insights
- Overall churn rate and state-wise distribution  
- Impact of international plan  
- Main churn reasons (competition & offers)  
- Demographic and usage-based churn analysis  

## 🛠 Tools
- Microsoft Excel  
- Pivot Tables & Charts  

## 🎯 Goal
Support data-driven decisions to reduce customer churn and improve retention.
